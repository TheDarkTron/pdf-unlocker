# PDF Unlocker

Python script to unlock password protected pdf files

## Dependencies

* python3
* pikepdf~=2.16.1

## Usage

```shell
unlockpdf [-h] files [files ...] password

Python script to unlock password protected pdf files

positional arguments:
  files       the pdf files to unlock
  password    the password

optional arguments:
  -h, --help  show this help message and exit
```
